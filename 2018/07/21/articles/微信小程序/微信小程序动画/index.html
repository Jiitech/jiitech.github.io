前端 微信小程序 动画 2018/07/21

插入一个小知识点

微信小程序中的多个空格怎么打？ &nbsp不行，需要使用一下方式：

（一）小程序动画基本api

微信小程序根据 wx.createAnimation(OBJECT)创建一个animation动画，然后调用该实例创建一个动画，最后通过动画实例的export方法导出动画数据传递给组件的animation属性。

OBJECT参数说明：

参数 类型 必填 默认值 说明
duration Integer 否 400 动画持续时间，单位ms
timingFunction String 否 linear 定义动画的效果
delay Integer 否 0 动画延迟时间，单位 ms
transformOrigin String 否 “50% 50% 0” 设置transform-origin
